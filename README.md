# www.ireviewstuff.com

This project is to create www.ireviewstuff.com, a personal webpage for Paul(ina) where she can showcase information about her and also upload and manage reviews from media she has consumed. It currently has API integrations to fetch music, movie, books and videogame information for the reviews. 

All full-stack work has been done by me!

## Working with:

* **React.js** using Next.js
* **Tailwind CSS** for front-end styling
* **Vercel** for hosting
* **Supabase** for DB management
* **Claude LLM** for resolving pesky problems
